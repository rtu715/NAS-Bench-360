A NAS Benchmark for Diverse Tasks


NAS-Bench-360 is a benchmark suite for evaluating state-of-the-art NAS methods on a diverse set of tasks. The selection spans different application domains, dataset sizes, problem dimensionalities, and learning objectives. Please contact Renbo Tu if you have questions or wish to contribute to the benchmark.
